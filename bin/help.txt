
Usage: doc <command> [options]

Commands:
   i, init        Initialize a project
   d, dev         Start preview service
   b, build       Build all documents

Options:
  -p, --path      Project root directory
  -t, --type      Project type (for init)
  -v, --version   Version information
  -h, --help      Help information
