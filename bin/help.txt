
Usage: doc <command> [options]

Commands:
  i, init         Initialize a Doczilla project
  d, dev          Dev current project
  b, build        Build current project

Options:
  -p, --path      Specify project path
  -v, --version   Display the current version of CLI
  -h, --help      Display cli help information

Examples:

  doc -p ./your-project-path
